A Balanced Perspective

Too much thought about the future brings fear. Too much thought about the past brings dissatisfaction with the present and fear of the future. Too much thought about the present brings an insatiable thirst for pleasure. To live a healthy life, one must be aware of past, present, and future circumstances.

Beautiful Women

There will never be a shortage of beautiful women in this world. Yesterday, when I wasn’t born yet, there were beautiful women in the world—I am sure of it. Today, when I am young and healthy, there are beautiful women. Tomorrow, when I am old, when too much excitement could bring me heart failure, when lack of self-control will hurt the one woman I will have committed to (not to mention our children and everyone who knows them), when I will be too weak to do the things I used to do with ease, guess what…there will still be beautiful women in this world.

So, if I don’t discipline myself now, if I don’t learn to ignore physical appearance and focus on good character, then I will be a very, very frustrated old man in the future; because there will never be a shortage of beautiful women in this world. I don’t like being frustrated, so I must ignore physical beauty.

Life and Recursion

If God were a computer programmer, the following could represent how He designed life on earth to be:

  • The simple version (boolean means a true or false value)
                                    boolean life_on_earth(Human parent1, Human parent2) {
                                    if (is_time_for_Armageddon()) {
                                        return false; //stop recursion…life on earth as we know it is over
                                    } else {
                                        Baby new_born = parent1.make_baby_with(parent2);
                                        return life_on_earth(new_born.grow_up(), new_born.find_suitable_partner()); 
                                    }
                                }
  • The complicated version (&& means “and” ... || means “or”)
                                    boolean life_on_earth_complex(Human person1, Human person2) {
                                    if (is_time_for_Armageddon()) {
                                        return false; //stop recursion
                                    } else if (person1.is_married_to(person2) && ( (person1.is_male() && person2.is_female())
                                                || (person1.is_female() && person2.is_male()) )) {
                                        bless_them_with_fertility();
                                        while (person1.is_willing() && person2.is_willing()) {
                                            Baby new_born = person1.make_baby_with(person2);
                                            return life_on_earth_complex(new_born. grow_up(), new_born.find_suitable_partner());
                                        }
                                    } else {
                                        return life_on_earth_complex(person1, person2.find_suitable_partner())
                                                && life_on_earth_complex(person2, person1.find_suitable_partner());
                                    }
                                }
